# **App Name**: StockPilot

## Core Features:

- Stock In: Record the details of incoming stock, including item name, quantity, and supplier.
- Stock Out: Record the details of outgoing stock, including item name, quantity, and recipient.
- Inventory Display: Display a current summary of inventory levels for each item.
- AI Chat Agent: Enable users to ask questions about current stock and the LLM agent decides if information should be extracted from a database. LLM replies will provide useful insights based on stock management principles using a tool.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and efficiency.
- Background color: Light gray (#F0F2F5) for a clean and minimal look.
- Accent color: Soft violet (#9575CD) to highlight interactive elements.
- Body and headline font: 'Inter', a sans-serif font known for its readability and neutral aesthetic.
- Use simple, outlined icons to represent different inventory actions.
- Clean, minimalist layout with a focus on readability and ease of navigation.
- Subtle transitions for page navigation and button clicks.
